Repository for the paper titled: "On the Feasibility of Exact Unitary Transformations for Many-body Hamiltonians"  
https://arxiv.org/abs/2510.10957  

Codes to perform and test unitary transformations based on generator eigenspectrum.  
See `samples.ipynb` for examples listed in Table 1.  

Python based, requires openfermion and standard python libraries: numpy, scipy, copy, itertools.  

## Thesis related (DO NOT CHANGE THIS SECTION)
- term_growth_avg.py - transform H4, 1A, sto-3g by random (seeded) UCC type generators. Archived results including run settings, generators, angles, etc are stored in ./thesis data
